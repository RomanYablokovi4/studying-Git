С помощью Git програмиисты и разработчики ориентируются в коде и отслеживают изменения. Git позвляет вернуть файлы в исходное состояние и видеть изменения, внесенные в определенный период. Разработчик выполняет разные команды (например, commit, push), а все изменения синхронизируются с центральныь репозитроием. Git - это система контроля версий, а GitHub - онлайн-сервис, по сути социальная сеть. Одна изи основных целей GitHub - быть единым местом для проектов с исходным кодом. Предполагается, что пользователь делится чем-то полезным, а другие люди смогут учавствовать в разработке.
